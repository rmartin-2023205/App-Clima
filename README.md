# App-Clima

**App-Clima** es una página web para consultar el clima actual y el pronóstico de 7 días de cualquier ciudad. Utiliza la API gratuita de **Open-Meteo** (sin necesidad de API key) e incluye funciones de favoritos y una interfaz moderna con efectos *glassmorphism*.

## 🚀 Funcionalidades

- 🔎 **Búsqueda por ciudad** con geocodificación.
- 🌤️ **Clima actual** (temperatura, sensación térmica, humedad, viento, descripción).
- 📅 **Pronóstico de 7 días** con tarjetas animadas.
- ⭐ **Favoritos** guardados en `localStorage`.
- 📱 **Diseño responsive** con menú móvil.
- ✨ **Animaciones y efectos visuales** (Scroll Reveal, glassmorphism, gradientes).
- 📬 **Formulario de contacto** con validación básica.

## 🧩 Tecnologías utilizadas

- **HTML5**
- **CSS3** + **TailwindCSS Play CDN**
- **JavaScript (ES6+)**
- **API Open-Meteo**
  - Geocoding: `geocoding-api.open-meteo.com`
  - Weather: `api.open-meteo.com`

## ⚙️ Cómo usar

1. Clona el repositorio:
   ```bash
   git clone https://github.com/rmartin-2023205/App-Clima.git
   ```
2. Abre `index.html` en tu navegador.
3. Busca una ciudad y disfruta del clima ☀️🌧️❄️

## 💾 Favoritos

Las ciudades favoritas se almacenan localmente en el navegador usando `localStorage`, por lo que se conservan entre sesiones en el mismo dispositivo.
